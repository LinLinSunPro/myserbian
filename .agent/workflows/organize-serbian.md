# Workflow: Organize Serbian Content

Follow this logic to place new materials into the correct folder.

## 1. Analyze the Content

Determine if the material is primarily a **Grammar Rule** or a **Vocabulary List/Story**.

## 2. Grammar Routing

If the content explains a rule or structure, place it in `e:\Antigravity\Serbian\Grammar\`:

* **Word Order**: Does it explain where to put words (e.g., *sam*, *se*)?
  * Target: `01_Word_Order`
* **Negation**: Does it explain how to say "no" or "not"?
  * Target: `02_Negation`
* **Questions**: Does it explain *Da li*, *Ko*, *Šta*, etc.?
  * Target: `03_Question_Words`
* **Present Verbs**: Does it explain verb endings (-m, -š, -mo)?
  * Target: `04_Present_Verbs`
* **Identity (To Be)**: Does it explain *sam, si, je* (I am, you are)?
  * Target: `05_Biti_To_Be`
* **Possession**: Does it explain *moj, tvoj* or Gender?
  * Target: `06_Possessives_My_Your`

## 3. Vocabulary Routing

If the content is a list of words or a story/text, place it in `e:\Antigravity\Serbian\Vocabulary\`:

* **Family**: Relatives, relationships (mama, brat, snaja).
  * Target: `Family`
* **Professions**: Jobs, Countries, Nationalities.
  * Target: `Professions`
* **Health**: Body parts, symptoms, feeling sick.
  * Target: `Health`
* **Basics**: Alphabet, Greetings, Numbers, Time.
  * Target: `Basics`

## 4. File Naming

* Use descriptive names (e.g., `negation_rules.md`, `family_story_julija.md`).
* Do not use generic names like `notes.md`.
